# Continuous Learning v2

When resolving an issue or being instructed by the user to "learn from this" or "write into experience":
1. Create a markdown file inside `.agents/instincts/` describing the error or workflow optimization.
2. Structure the file with frontmatter (description, triggers, confidence) and a body clearly contrasting the BAD pattern vs the GOOD pattern.
