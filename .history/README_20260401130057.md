# 肺结节随访建议生成智能体

本项目面向肺癌高风险人群筛查与随访管理场景，目标是构建一个基于因果解耦与神经符号推理的肺结节随访建议生成系统。

## 项目目标
- 从非结构化电子病历和影像报告中提取高保真临床事实
- 将“重度吸烟”等定性描述映射到指南中的定量标准
- 基于知识图谱与神经符号推理生成符合指南约束的随访建议
- 输出包含随访建议、推理过程和指南依据的标准化报告

## 项目结构
- `docs/`：文档与设计说明
- `src/`：核心程序
- `scripts/`：脚本工具
- `tests/`：测试代码
- `notebooks/`：实验记录
- `data/`：数据目录（不上传敏感原始数据）
